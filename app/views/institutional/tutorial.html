Manual de instalación

Configura ClaveÚnica en tu servicio

Luego de que ya tienes el Cliente ID y Secret ID y nosotros tu URI de Redirección, ya estarás listo para montar tu servicio con ClaveÚnica.

El flujo soportado por ClaveÚnica es Authorization code Flow por lo que explicaremos el proceso de configuración en estos 6 pasos:

Paso 1
Crear Token de estado anti-falsificación
Paso 2
Enviar una solicitud de autenticación al servicio de ClaveÚnica
Paso 3
Confirmar el Token de estado de anti-falsificación
Paso 4
Cambiar el código de acceso por un Token de Acceso y un Token de autorización
Paso 5
Autenticar Usuario
Paso 6
Obtener información de ciudadano y Paso a Producción
